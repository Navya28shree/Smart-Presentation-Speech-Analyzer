# Smart Presentation Speech Analyzer

Smart Presentation Speech Analyzer is a web-based AI tool that helps users improve their presentation skills. It records voice input, converts speech to text, and analyzes the speaking style to provide feedback on confidence, nervousness, and clarity.

## Features

* Record voice directly in the browser
* Automatic speech-to-text conversion
* AI-based speech analysis
* Detection of filler words and nervousness
* Suggestions for improvement

## Technologies Used

* HTML
* CSS
* JavaScript
* Python (Flask)
* Groq API
* Speech Recognition

## Use Cases

* Students practicing presentations
* Placement and interview preparation
* Public speaking improvement
* Seminar practice

## How to Run

1. Clone the repository
2. Install the required Python packages
3. Run the Flask application
4. Open the browser and start recording
